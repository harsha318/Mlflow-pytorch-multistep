# AIOps-mlflow-pytorch-multistep-workflow
AIOps-mlflow-pytorch-multistep-workflow

## STEPS -

### STEP 01- Clone this new repository

### STEP 02- run init_setup.bash to create environment
```bash 
bash init_setup.sh
```

### STEP 03- Activate the new environment - 
```bash
conda activate ./env
```

### STEP 03- to run this project -
```bash
mlflow run . --no-conda
```
